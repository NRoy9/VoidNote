FEATURES

Everything you need.
Nothing you don't.

Void Note is packed with premium features — for free. No paywalls on core functionality. Ever.

✍
RICH TEXT EDITOR

Bold, italic, underline, strikethrough. H1/H2/H3 headings. Numbered lists with auto-continue. Interactive checklists. Color-coded notes. All the formatting tools you need — none of the bloat.

🔐
AES-256-GCM ENCRYPTION

Every note is encrypted before it touches storage. Master key lives in Android Keystore (hardware-backed). Zero-knowledge by design.

👁
BIOMETRIC LOCK

Fingerprint or face unlock. Vault password as fallback. Your notes open only for you.

📁
FOLDERS + TAGS

Organise with nested folders and up to 5 tags per note. Browse by tag, folder, or search everything at once.

🎙
MEDIA BLOCKS

Attach encrypted voice recordings and images directly to notes. Captured from camera or gallery. All files encrypted at rest.

✈
100% OFFLINE-FIRST

No network required. Ever. Your data never leaves your device unless you explicitly choose to export it. No accounts needed for core features.

🌑
DARK · LIGHT · EXTRA DARK (OLED)

Three carefully crafted themes. Extra Dark mode uses true #000000 black for OLED screens — saves battery and looks stunning. Nothing-inspired monochromatic design throughout. Note color accents (6 colours) add personal touches without breaking the aesthetic.

💾
ENCRYPTED BACKUPS

Export your entire vault as an encrypted .vnbackup file. Restore on a new device. Your encryption key never leaves your device.

🗂
ARCHIVE + TRASH

Archive notes without deleting. Trash with 30-day auto-cleanup. WorkManager runs the cleanup in the background — no manual purging needed.

↕
SORT & FILTER

Sort by last modified, created date, or title A→Z / Z→A. Filter chips for All, Pinned, Folders, and Tags. Sort preference persists across sessions.

AES-256-GCM ANDROID KEYSTORE PBKDF2 ZERO-KNOWLEDGE

Security that's
not optional.

Privacy isn't a feature toggle. It's the foundation. Void Note was designed encryption-first — your notes are encrypted before they are ever written to disk.

🔑
ANDROID KEYSTORE MASTER KEY

The master encryption key lives in Android's hardware-backed Keystore. It never exists in plain memory longer than needed.

🛡
AES-256-GCM NOTE ENCRYPTION

Every note title, content, and tag is encrypted individually. GCM mode provides authenticated encryption — tampered data is detected.

🧂
PBKDF2 KEY DERIVATION

Your vault password is never stored. It's run through PBKDF2 to derive the encryption key. The salt is stored — the password never is.

📁
ENCRYPTED MEDIA FILES

Images and audio recordings are encrypted as .enc files in app-private storage. No unencrypted files are ever written to disk.

VOID NOTE — ENCRYPTION PIPELINE
// User creates a note
val note = Note(title = "My secret", ...)
 
// Fetch master key from Android Keystore
val masterKey = keystore.getOrCreate()
 
// Encrypt title + content with AES-256-GCM
val encrypted = aes256gcm.encrypt(
  plaintext = note.content,
  key = masterKey
)
 
// Only the cipher text reaches Room DB
db.noteDao().insert(NoteEntity(
  content = encrypted.cipherText, // ← stored
  iv = encrypted.iv, // ← stored
  // plaintext never stored ✓
))
 
✓ Note encrypted and saved.
