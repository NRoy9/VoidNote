2. Description of the App

Void Note is a privacy-focused note-taking application for Android that provides:

  • Encrypted local note storage with AES-256-GCM encryption
  • Rich text editing with formatting, checklists, and media attachments
  • Vault-based security with biometric and password authentication
  • Folder and tag-based note organisation
  • Import/export functionality for note backup and portability
