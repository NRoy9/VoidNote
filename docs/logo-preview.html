DESIGN RATIONALE

The logo has three layers of meaning stacked in one mark:

Document shape — the folded-corner page is universally understood as "notes/document" at a glance, even at 20px.

Red fold — the only red element besides the void circle. The fold is where a physical note gets "dog-eared." The red makes it immediately eye-catching and matches VoidAccent = Color(0xFFFF3B30) exactly from your app's Color.kt.

Void circle — the ○ from the website is preserved inside the document, bottom-right. It ties the brand together across website and app. "The void is inside the note."

Consistency: This exact SVG code goes into the website nav, Android XML drawable, and Play Store 512×512 PNG export — one source of truth.
